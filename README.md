# Compiling & Runing the game 
Clone the repository, open your terminal and run the following command: 
``` gcc -o main.exe main.c game_interface.c game.c `sdl-config --libs` -lSDL_image ```

``` ./main.exe ``` 

# Requirements 

✔ SDL 1.2 
✔ SDL_Images 1.2
